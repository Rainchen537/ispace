# handsbnbu GitHub Pages

这个仓库用于托管“掌上BNBU”的公开隐私政策页面。

## 本地文件

- `index.html`: 隐私政策主页

## 发布到 GitHub Pages

1. 在 GitHub 上创建一个公开仓库。
2. 把这个目录内容推送到该仓库的 `main` 分支。
3. 打开 GitHub 仓库 `Settings` -> `Pages`。
4. 在 `Build and deployment` 中选择 `Deploy from a branch`。
5. 选择 `main` 分支和 `/ (root)` 目录。
6. 等待 GitHub Pages 发布完成。

发布后的 URL 可能类似于：

- `https://<你的 GitHub 用户名>.github.io/<仓库名>/`
- 如果仓库名本身就是 `<你的 GitHub 用户名>.github.io`，则 URL 会是 `https://<你的 GitHub 用户名>.github.io/`

## App Store Connect

把发布后的 HTTPS 链接填到：

- `Privacy Policy URL`
- 如果需要，也可以同一个链接同时填到 `Support URL`
